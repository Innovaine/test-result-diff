# test-result-diff

Warehouse for an AI agent company project, synced from **Qadar**.

## What's in here

- `week-NN/` — Weekly artifacts produced by the agents (engineering files, design HTML/SVG, PM status, code reviews, finance, risks).
- `customer-messages.md` — Real customer responses, captured by the project owner. Empty = no signal yet.
- Project-level metadata + planner briefs.
